Email Deliverability Agent

This agent analyzes your email setup and identifies why your emails are not reaching the inbox — including authentication, reputation, and sending issues.

Step 1: Enter Your Domain

The agent will check SPF, DKIM, and DMARC records for your sending domain.

(Automation coming soon — this agent will auto-detect DNS records.)

What This Agent Checks

  • SPF record presence and alignment
  • DKIM selector existence
  • DMARC policy configuration
  • Common authentication failures
  • Inbox vs spam risk signals

Agent Actions (Planned)

  • Detect missing or broken DNS records
  • Generate correct SPF / DKIM / DMARC values
  • Provide copy-paste DNS fixes
  • Offer auto-fix via provider APIs (advanced)

This is part of the Fix Library — evolving from static fixes into automated agents.
